# Git First Project — Engineering Log

This repository started as a Git learning project and has evolved into a
daily engineering log.

## Purpose
- Practice Git and GitHub workflows
- Maintain daily development consistency
- Document learnings, mistakes, and insights
- Store small experiments and code snippets

## Structure
- logs/        → Daily learning logs
- snippets/    → Reusable code fragments
- experiments/ → Small tests and practice work

This repository is intentionally simple and continuously evolving.
