## ⚙️ Configuration

### GitHub CLI
- Authenticate with `gh auth login`.
- Use `gh repo create git-first-project --public --source=. --remote=origin --push` to publish from terminal.

### Recommended Git config
```bash
git config --global user.name "LegendarySumit"
git config --global user.email "your-email@example.com"
git config --global init.defaultBranch main
```

### Git workflow
- `git switch -c feature/<name>` — create and switch to a new branch
- `git switch main` — return to the main branch
- `git merge feature/<name>` — merge work into main

## 🐛 Troubleshooting

- **`git: unknown command 'switch'`** — update Git to a newer version.
- **Conflicts during merge** — resolve files, then `git add <file>` and `git commit`.
- **Python errors** — check indentation, loop conditions, and string method calls.
