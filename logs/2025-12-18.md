**Java Input & Output Streams**

- **`System.in`**
  - Standard input stream
  - Used for taking input from the keyboard

- **`System.out`**
  - Standard output stream
  - Used for displaying normal program output

- **`System.err`**
  - Standard error stream
  - Used for error messages and debugging

**Key Insight:** Separating output and error streams is useful in real applications for debugging and logging.

**Type Casting**

- Understood **implicit (widening)** type casting
- Understood **explicit (narrowing)** type casting
- Learned when and why casting is required
- Recognized potential data loss during narrowing conversions

**Variables in Java**

Learned the meaning, scope, and usage of:

- **Local Variables**
  - Declared inside methods or blocks
  - Accessible only within that scope

- **Instance Variables**
  - Belong to an object
  - Each object has its own copy

- **Static Variables**
  - Belong to the class
  - Shared across all objects

**Key Takeaway:** Variable type directly affects memory usage and behavior.
