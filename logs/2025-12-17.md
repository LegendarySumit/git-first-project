### üß† Main Method Deep Dive
```java
public static void main(String[] args) {
    // Program entry point
}
```

**Keyword Understanding:**

| Keyword | Purpose |
|---------|---------|
| `public` | Accessible by the JVM from anywhere |
| `static` | Can be called without creating an object |
| `void` | Does not return any value |
| `main` | Entry point recognized by the JVM |
| `String[] args` | Accepts command-line arguments |

### üí° Key Insights

- Understanding **why** keywords are used, not just memorizing them
- Clear mental model of Java's execution flow
- Importance of platform independence in modern development
