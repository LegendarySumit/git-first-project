## ğŸ§ª Experiment 2 â€” Algorithm vs Program

### Observation

**Algorithm:**
- Logical steps to solve a problem
- Language-independent
- Abstract representation

**Program:**
- Implementation of algorithm in a specific language
- Language-dependent
- Concrete implementation

### Example

**Algorithm:** Steps to add two numbers
```
1. Input: a, b
2. sum = a + b
3. Output: sum
4. Stop
```

**Program:** Java implementation
```java
int a = 5, b = 10;
int sum = a + b;
System.out.println(sum);
```

### Takeaway
âœ… Algorithm is **language-independent** â€” same logic, different implementations.
